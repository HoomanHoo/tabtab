# 프로젝트 개요

약국 재고 관리 서비스 tabtab 입니다.
현 약국 재고 관리 어플리케이션은 난잡한 UI와 사용자의 PC에 설치하여 사용하는 형태로 보급률이 낮은 상황입니다. 
따라서 저희는 웹 기반으로 간편하게 사용할 수 있으며 세련된 UI를 목표로 tabtab 서비스를 만들고자 하였습니다.
